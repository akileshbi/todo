## Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code with Flutter plugins

### Installation

1. Clone this repository:
```bash
git clone https://github.com/akilesh-pvt/todo-app.git
cd todo-app
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
